# Skill: trend_analysis

## goal
Analyze research evolution for a keyword and time range.

## input
- keyword
- paper list
- years

## output
- timeline
- main technical routes
- common backbones or modules
- hot datasets
- hot metrics
- evolution trend

## instruction
Group papers by year and method tags. Extract repeated architecture tokens, datasets, and metrics from metadata. Use LLM synthesis when available.
